# Technology Stack

- Ruby 2.6.4
- Ruby on Rails 6.0.2
- PostgreSQL 11
- React 17

# Get started

- Clone the repository:

  ```sh
  git clone git@github.com:9jaswag/finder.git
  ```

- Ensure you have PostgreSQL on your computer.
- Set up database:
  ```sh
  rails db:setup
  ```
- Serve the app:
  ```sh
  rails server
  ```
- View the app in the browser at http://localhost:3000/
